# WTT Custom Quests - TLQ (Tarkov Lore Quests)

Кастомные квесты на русском языке для Escape from Tarkov на основе **WTT-ServerCommonLib**.

---

## 📋 Описание

Мод добавляет **кастомные квесты** для торговцев игры, включая:

### Ragman
- **"Полосатое животное"** — 10 последовательных квестов на устранение Killa и сбор коллекции

### Peacekeeper
- **"Ночные операции"** — 5 частей (устранение Решалы и свиты)
- **"Поздний час"** — 1 квест

### Prapor
- **"Очень плохая привычка"** — 3 части (алкоголь и табак)
- **"Байки"** — 2 части (анекдоты)
- **"Бумажник"** — 1 квест
- **"Подрядчик"** — 2 части (тактическое снаряжение)

### Fence
- **"Начинающий коллекционер"** — 1 квест
- **"Медпоставка"** — 1 квест
- **"Очищение"** — 2 части (ЧВК)

### Skier
- **"Хозяин района"** — 1 квест

### Jaeger
- **"Отстрел диких"** — 1 квест

## 🔧 Зависимости

Для работы мода **ОБЯЗАТЕЛЬНО** требуется:

### WTT-ServerCommonLib
- **Ссылка:** [https://forge.sp-tarkov.com/mod/2310/wtt-commonlib](https://forge.sp-tarkov.com/mod/2310/wtt-commonlib)
- **Файл:** `WTT-ServerCommonLib.dll`
- **Установка:** Скопируйте DLL в папку `BepInEx/plugins/`

**Без этой библиотеки мод НЕ будет работать!**

---

## 📁 Установка

1. Установите **BEpInEx** (если ещё не установлен)
2. Установите **WTT-ServerCommonLib**
3. Скопируйте содержимое папки `db/` в корень мода в `BepInEx/plugins/TLQ/`
4. Запустите игру

**Структура папок:**
```
BepInEx/plugins/TLQ/
├── db/
│   └── CustomQuests/
│       ├── Ragman/
│       ├── Peacekeeper/
│       ├── Prapor/
│       ├── Fence/
│       ├── Skier/
│       ├── Jaeger/
│       └── Therapist/
└── TLQ.dll
```
